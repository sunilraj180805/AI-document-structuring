# AI-document-structuring
1. Install Dependencies:
   pip install google-generativeai pandas PyPDF2 openpyxl jupyter
   
2. Set Your Gemini API Key:
   Create an environment variable named GOOGLE_API_KEY.
   Add your key.
   
3. Open the Project:
   Set the input file and give.
   Open soln.ipynb in jupyter notebook and run all the cells.

4. Retrieve the Final Output:
   The output will be saved as output.xlsx.
